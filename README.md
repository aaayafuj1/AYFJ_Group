## Quick Start

Clone any repo and run locally. Example for a Python tool (Python **3.10+** recommended):

```bash
# 1) clone
   git clone https://github.com/AYFJ-Group/ayfj-utils.git
   cd ayfj-utils

# 2) optional: create virtual env
 python3.10 -m venv .venv
 source .venv/bin/activate  # Windows: .venv\\Scripts\\activate

# 3) install deps
 pip install -r requirements.txt

# 4) run
 python -m ayfj
```

Example for a .NET tool:

```bash
 dotnet build -c Release
 ./bin/Release/*/publish/ayfj-tool
```

> If a project requires Kali/Ubuntu packages, list them in that project’s README under **Requirements**.

## Security

Found a vulnerability in an AYFJ project?

* Please **do not** open a public issue with exploit details.
* Email the maintainers (see **Contact**) with a minimal reproducible report.
* We follow **responsible disclosure** practices. Patches come first, then advisories.

## Illegal & Unethical

**All code and content in AYFJ projects are for Awareness and defensive purposes only.**

* You must have **explicit permission or without** to test or assess any system.
* Authors and contributors are **not responsible** for misuse.
* Some repositories may include simulated or containerized labs — these are isolated learning environments and **must not** be pointed at real targets.

If you’re unsure whether something is permitted, assume **it is not** and ask first.
